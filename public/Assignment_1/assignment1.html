Section 1 - Comparing Categories

Data visualization for 游꺕 5M Trees Dataset

In this assignment we will use the 5M Trees Dataset from Kaggle.
It contains data about 5 million trees from 63 US cities.
Our purpose is to analyze the position, the quantity, the scientific name and the common name of the trees.

Data preparation

For all the datasets we used in the visualizations we extracted the 'city', 'scientific_name' and 'state' columns from the original dataset and dropped any rows with missing values, resulting in a total number 5130468 of rows.

  1. 游릭 We extracted the 'state' column from the merged dataset. Next, we counted the occurrences of each unique state in the 'state' column and sorted them in descending order to identify the top 20 states with the most trees. We saved this refined dataset to a CSV file named "Top20_states_MostTrees.csv".

  2. 游릭 We extracted the 'city' column from the merged dataset with the information of all the cities file. Next, we counted the occurrences of each unique city in the 'city' column and sorted them in descending order to identify the top 20 cities with the most trees. We saved this refined dataset to a CSV file named "Top20_city_MostTrees.csv".

  3. 游릭 First, we extracted the 'city' and 'scientific_name' columns from the original dataset. Then, we calculated the total counts of each tree type in each city. Next, we filtered the cities where all three selected tree types (Platanus acerifolia, Lagerstroemia indica, and Acer platanoides) exist. We created separate DataFrames for each tree type (Platanus acerifolia, Acer platanoides, and Lagerstroemia indica) and renamed the count columns accordingly. We filtered these DataFrames to include only the cities where all three tree types are present. Additionally, we created an 'Other' DataFrame to account for tree types other than the three selected. To combine the information, we merged the individual tree type DataFrames and the 'Other' DataFrame based on the 'city' column. We saved the resulting dataset to a CSV file named "Top15_city_3Tree_others.csv".

  4. 游릭 Just like the previous dataset, we extracted the 'city' and 'scientific_name' columns from the original dataset. We calculated the total counts of each tree type in each city and we filtered the cities where all three tree types exist. We also created an 'Other' DataFrame to account for tree types other than the three selected. We calculated the total count of all tree types in each city and we sorted them in descending order. We created separate files for the top 5, top 10, and top 15; saved to CSV files named "Top5_3Tree&others.csv", "Top10_3Tree&others.csv" and "Top15_3Tree&others.csv".

Vertical Barchart

Our first visualization is a vertical barchart. It shows the twenty states of US with the highest number of trees indeed. The x-axis displays the names of the states while the y-axis represents the number of trees. Each bar on the chart corresponds to the tree count for a specific US state.
For this visualization we opted to focus on states rather than cities to provide a more comprehensive and overarching perspective of the most densely forested regions in the US.

Horizontal Barchart

Our second visualization is a horizontal barchart. It shows the twenty cities of US with the highest number of trees. The x-axis displays the names of the cities while the y-axis represents the number of trees. Each bar on the chart corresponds to the tree count for a specific US city.
Differently from the first chart, we have chosen to concentrate on cities in this instance to offer a more localized perspective. The reason behind this decision is to provide a finer-grained analysis of tree distribution within urban areas.
We can notice that the vertical and the horizontal barcharts are coherent indeed in the first chart, California emerges as the most tree-abundant state, while in the second graph, Los Angeles, a city within the state of California, stands out as the city with the highest number of trees. The same hold for New York.

Stacked Barchart

Our third visualization is a stacked bar chart. It highlights three distinct tree types: Acer Platanoides (Norway Maple), Lagerstroemia Indica (Crape Myrtle), and Platanus Acerifolia (London Plane) across multiple cities. We selected these three tree species because they were the most prevalent and allowed us to compare the tree populations in various cities in fact we've chosen the top 15 cities with the highest counts of these tree types.
The x-axis displays the names of the cities while the y-axis represents the total number of trees. Each bar's height corresponds to the total number of trees in a particular city, and the segments in it represent the distribution of the three specified tree types within each city.

Small Multiple Barchart

Our fourth visualization is a small multiple barchart. It highlights three distinct tree types: Acer Platanoides (Norway Maple), Lagerstroemia Indica (Crape Myrtle), and Platanus Acerifolia (London Plane) across the top 15 cities with the highest counts of these tree types (as in the previous chart).
This visualization is composed of four different horizontal barcharts, one for each tree type and one for all the other tree types. The x-axis of each barchart displays the number of trees while the y-axis displays the names of the cities. Each bar in each barchart corresponds to the quantity of a specific tree type in a specific city.
The length of the bars in each barchart is computed by normalizing the original values such that the scale of the x-axis of the plots is the same for all of them thus allowing for a direct comparison between the different tree types.
Essentially the data displayed in this graph is the same as that displayed in the stacked barchart above, however this type of visualization in some cases can be clearer and more intuitive because more information can be obtained at a glance.

Percent Stacked Barplot

Our fifth visualization is a percent stacked barplot. It highlights three distinct tree types: Acer Platanoides (Norway Maple), Lagerstroemia Indica (Crape Myrtle), and Platanus Acerifolia (London Plane) across the top 15 cities with the highest counts of these tree types (like the previous two charts).
The x-axis displays the names of the cities while the y-axis represents the total number of trees as a percentage. Each city is represented by a 100% bar, where the entire bar represents the total number of trees while each segments represents the percentage of a specific tree types and the remaining other tree types in the city.
This type of visualization is good for showing the composition of a whole in terms of its individual components. The bar representation as a 100%, and segments within the bars show the proportions of different categories or subgroups, making it easy to compare how these subgroups contribute to the whole.

Heatmap

Our sixth visualization is a heatmap. As the previous visualizations, it highlights three distinct tree types: Acer Platanoides (Norway Maple), Lagerstroemia Indica (Crape Myrtle), and Platanus Acerifolia (London Plane) across multiple cities.
The x-axis displays the names of the cities, the y-axis represents the different categories of trees and the squares in the plot represent the number of trees using color gradients (darker color if the quantity of trees is high and a lighter color if the quantity of trees is low).
We noticed that one single plot for the 15 cities with the highest counts of these tree types was not good for conveying information since the cities with a small amount of trees were about of the same color, for this reason we decided to give the possibility to choose, with a dropdown menu, whether to display the top 5, 10 and 15 cities with the highest counts of these tree types.
