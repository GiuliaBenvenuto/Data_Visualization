Section 4 - Maps

Spatial data visualization of the trees in the United States

In this assignment we will use the 5M Trees Dataset from Kaggle in order to analyze the spatial distribution of the trees in the United States.

Data preparation

Firstly we loaded the 5MTree Dataset, we delete all the rows with incomplete information. Doing so the number of states in our data decresed to 23 states in total.
Due to the huge amount of data at our disposal we decided to group near trees with the same scientific name and simiar location. While this help us reduce the dimension of the data, the number of data is still high amounting to 46435 rows.
Additionally, to compute the density of tree for each state, we had to insert by hand the area of each state.

Dot density map

Only one symbol, only one color

Our first visualization is a dot density map with only one symbol and only one color.
Initially we noticed that the total number of trees was 3912954, but the visualization of them was impossible because the size of the dataset was too big. Thus we decided to group together trees with the same scientific name that were in similar locations basing on latitude and longitude coordinates saving the number of trees in each group and using it to give a radius to the circle representing the group.

Dot density map

Only one symbol, several colors

Our second visualization is a dot density map with only one symbol and several colors.
For the displacement of the dots and for their radius we made the same reasonment as the previous chart.
In this case we also selected the top 10 most common scientific names and we assigned to them a specific color which is used to color the dots.

Choropleth map

Tree abundance

Our third visualization is a choropleth map.
In this case the chart shows the tree abundance in each state, this information is conveyed by the color of the states.
We can notice that the state with the highest number of trees is California indeed its color is the darkes one and we can also notice that there are some states without available data due to the incompleteness of the dataset which are colored with a light grey.

Choropleth map

Tree density

Our fourth visualization is a choropleth map.
In this case the chart shows the tree density in each state, again, this information is conveyed by the color of the states.
The density value is computed by dividing the number of trees in each state by the area of the state.
